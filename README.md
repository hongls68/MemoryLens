# MemoryLens (记忆透镜) 项目介绍

## 1. 产品概述
**产品名称**：MemoryLens (记忆透镜)  
**核心 Slogan**：独居老人的“第二大脑”：跌倒秒级报警，遗忘瞬间回溯。

MemoryLens 是一款专为独居老人设计的智能穿戴辅助系统，旨在通过 AI 技术解决安全监测、记忆辅助及情感陪伴三大核心问题。

---

## 2. 用户痛点分析
当前独居老人及家属面临的主要挑战包括：

- **安全盲区**  
  老人跌倒后往往无法自救，而传统手环误报率高且缺乏视频确认机制，导致救援延迟。
  
- **记忆断片**  
  轻度认知障碍老人常出现短期记忆丢失（如忘记钥匙位置、是否服药），子女无法远程协助回忆，造成生活困扰。
  
- **情感孤独**  
  缺乏基于实时场景的主动式自然对话陪伴，老人精神世界空虚。

---

## 3. 解决方案

### 3.1 核心功能
1. **第一视角危险预警**  
   通过佩戴式摄像头实时监测跌倒、火灾或异常静止状态。一旦检测到危险，系统在 **5秒内** 推送现场视频警报给子女。
   
2. **视觉记忆回溯**  
   支持自然语言语音交互（例如：“我刚才把眼镜放哪了？”）。AI 自动检索过去 1 小时的视频流，精准定位物体位置并反馈给用户。
   
3. **场景化主动交互**  
   系统能识别特定行为场景（如老人对着药瓶发呆），并主动发起语音提醒，指导服药步骤，变被动响应为主动关怀。

### 3.2 AI 技术栈
本项目采用前沿的多模态 AI 架构：
- **多模态大模型 (MLLM)**：如 Qwen-VL-2026，用于深度理解视频场景与物体间的复杂关系。
- **端侧小模型 (SLM)**：部署在低功耗芯片上，本地运行跌倒检测算法，确保数据隐私与低延迟。
- **RAG (检索增强生成)**：构建个人短期记忆库，实现对历史视频内容的精准问答与回溯。
